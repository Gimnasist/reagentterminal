Tool to control and account reagnets at lab with automatic scale reading and barcode managing
Utilizes PyQt 5.4.2 and MySQL database as storage place for reagent data.
Needs configuration of manufacturer of your COM-adapter in weightingReagentDlg.py for being able to run.
Test on Raspberry Pi 2 (Raspbian Whezzy, kernel 4.0.7) as host machine.
